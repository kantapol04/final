ยินดีต้อนรับเข้าสู่เว็บไซต์เพื่อการศึกษาสำหรับการเรียนการสอน วิชาพื้นฐานการสร้างเว็บไซต์

คำอธิบายรายวิชา ศึกษาและปฏิบัติเกี่ยวกับโครงสร้างของเว็บไซต์ หลักการออกแบบเว็บไซต์ด้วยโปรแกรมภาษา HTML5 การตกแต่งข้อความ การแทรกรูปภาพหรือสื่อต่างๆ การสร้างตาราง การสร้าง
การเชื่อมโยงในแบบต่างๆ การสร้างเฟรม การออกแบบและสร้างฟอร์ม การแทรก Java Script การทดสอบการทำงานของเว็บไซต์ การอัพโหลดเว็บไซต์ไปยังเว็บเซิร์ฟเวอร์
